# 项目文件目录说明
1.bin           ----- 用于应用启动
2.conf			----- 配置文件目录
3.model			----- 模型成，与数据库交互
4.node-modules	----- node模块文件目录
5.public		----- 静态资源文件目录
6.routes		----- 业务实际代码处理目录
7.util			----- 工具方法目录
8.views			----- 模板视图文件目录
9.app.js		----- 程序main文件
10.package.json ----- 项目信息配置和依赖模块管理文件



# 项目开发版本
node     v4.4.7
express  4.13.4
